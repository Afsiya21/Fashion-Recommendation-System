# Fashion Recommendation System

## Project Description

This project recommends similar fashion products using Machine Learning and Natural Language Processing (NLP).

## Technologies Used

- Python
- Google Collab
- Pandas
- NumPy
- Scikit-learn
- NLTK

## Dataset

Myntra Fashion Products Dataset

## Algorithm

1. Data Preprocessing
2. NLP
3. TF-IDF Vectorization
4. Cosine Similarity
5. Recommendation System

## How to Run

1. Open Fashion_Recommendation_System.ipynb in Google Collab.
2. Upload myntra_products_catalog.csv.
3. Run all cells.
4. Enter a product or brand name.
5. View recommended products.
